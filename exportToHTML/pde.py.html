2    This module contains pdsolve() and different helper functions that it 
3    uses. It is heavily inspired by the ode module and hence the basic 
4    infrastructure remains the same. 
5     
6    **Functions in this module** 
7     
8        These are the user functions in this module: 
9     
10       - pdsolve()     - Solves PDE's 
11       - classify_pde() - Classifies PDEs into possible hints for dsolve(). 
12       - pde_separate() - Separate variables in partial differential equation either by 
13                          additive or multiplicative separation approach. 
14    
15       These are the helper functions in this module: 
16    
17       - pde_separate_add() - Helper function for searching additive separable solutions. 
18       - pde_separate_mul() - Helper function for searching multiplicative 
19                              separable solutions. 
20    
21   **Currently implemented solver methods** 
22    
23   The following methods are implemented for solving partial differential 
24   equations.  See the docstrings of the various pde_hint() functions for 
25   more information on each (run help(pde)): 
26    
27     - 1st order linear homogeneous partial differential equations 
28       with constant coefficients. 
29     - 1st order linear general partial differential equations 
30       with constant coefficients. 
31     - 1st order linear partial differential equations with 
32       variable coefficients.

63   def pdsolve(eq, func=None, hint='default', dict=False, solvefun=None, **kwargs):
64       """ 
65       Solves any (supported) kind of partial differential equation. 
66    
67       **Usage** 
68    
69           pdsolve(eq, f(x,y), hint) -> Solve partial differential equation 
70           eq for function f(x,y), using method hint. 
71    
72       **Details** 
73    
74           ``eq`` can be any supported partial differential equation (see 
75               the pde docstring for supported methods).  This can either 
76               be an Equality, or an expression, which is assumed to be 
77               equal to 0. 
78    
79           ``f(x,y)`` is a function of two variables whose derivatives in that 
80               variable make up the partial differential equation. In many 
81               cases it is not necessary to provide this; it will be autodetected 
82               (and an error raised if it could not be detected). 
83    
84           ``hint`` is the solving method that you want pdsolve to use.  Use 
85               classify_pde(eq, f(x,y)) to get all of the possible hints for 
86               a PDE.  The default hint, 'default', will use whatever hint 
87               is returned first by classify_pde().  See Hints below for 
88               more options that you can use for hint. 
89    
90           ``solvefun`` is the convention used for arbitrary functions returned 
91               by the PDE solver. If not set by the user, it is set by default 
92               to be F. 
93    
94       **Hints** 
95    
96           Aside from the various solving methods, there are also some 
97           meta-hints that you can pass to pdsolve(): 
98    
99           "default": 
100                  This uses whatever hint is returned first by 
101                  classify_pde(). This is the default argument to 
102                  pdsolve(). 
103   
104          "all": 
105                  To make pdsolve apply all relevant classification hints, 
106                  use pdsolve(PDE, func, hint="all").  This will return a 
107                  dictionary of hint:solution terms.  If a hint causes 
108                  pdsolve to raise the NotImplementedError, value of that 
109                  hint's key will be the exception object raised.  The 
110                  dictionary will also include some special keys: 
111   
112                  - order: The order of the PDE.  See also ode_order() in 
113                    deutils.py 
114                  - default: The solution that would be returned by 
115                    default.  This is the one produced by the hint that 
116                    appears first in the tuple returned by classify_pde(). 
117   
118          "all_Integral": 
119                  This is the same as "all", except if a hint also has a 
120                  corresponding "_Integral" hint, it only returns the 
121                  "_Integral" hint.  This is useful if "all" causes 
122                  pdsolve() to hang because of a difficult or impossible 
123                  integral.  This meta-hint will also be much faster than 
124                  "all", because integrate() is an expensive routine. 
125   
126          See also the classify_pde() docstring for more info on hints, 
127          and the pde docstring for a list of all supported hints. 
128   
129      **Tips** 
130          - You can declare the derivative of an unknown function this way: 
131   
132              >>> from sympy import Function, Derivative 
133              >>> from sympy.abc import x, y # x and y are the independent variables 
134              >>> f = Function("f")(x, y) # f is a function of x and y 
135              >>> # fx will be the partial derivative of f with respect to x 
136              >>> fx = Derivative(f, x) 
137              >>> # fy will be the partial derivative of f with respect to y 
138              >>> fy = Derivative(f, y) 
139   
140          - See test_pde.py for many tests, which serves also as a set of 
141            examples for how to use pdsolve(). 
142          - pdsolve always returns an Equality class (except for the case 
143            when the hint is "all" or "all_Integral"). Note that it is not possible 
144            to get an explicit solution for f(x, y) as in the case of ODE's 
145          - Do help(pde.pde_hintname) to get help more information on a 
146            specific hint 
147   
148   
149      Examples 
150      ======== 
151   
152      >>> from sympy.solvers.pde import pdsolve 
153      >>> from sympy import Function, Eq 
154      >>> from sympy.abc import x, y 
155      >>> f = Function('f') 
156      >>> u = f(x, y) 
157      >>> ux = u.diff(x) 
158      >>> uy = u.diff(y) 
159      >>> eq = Eq(1 + (2*(ux/u)) + (3*(uy/u)), 0) 
160      >>> pdsolve(eq) 
161      Eq(f(x, y), F(3*x - 2*y)*exp(-2*x/13 - 3*y/13)) 
162   
163      """
164  
165      if not solvefun:
166          solvefun = Function('F')
167  
168      # See the docstring of _desolve for more details.
169      hints = _desolve(eq, func=func, hint=hint, simplify=True,
170                       type='pde', **kwargs)
171      eq = hints.pop('eq', False)
172      all_ = hints.pop('all', False)
173  
174      if all_:
175          # TODO : 'best' hint should be implemented when adequate
176          # number of hints are added.
177          pdedict = {}
178          failed_hints = {}
179          gethints = classify_pde(eq, dict=True)
180          pdedict.update({'order': gethints['order'],
181                          'default': gethints['default']})
182          for hint in hints:
183              try:
184                  rv = _helper_simplify(eq, hint, hints[hint]['func'],
185                      hints[hint]['order'], hints[hint][hint], solvefun)
186              except NotImplementedError as detail:
187                  failed_hints[hint] = detail
188              else:
189                  pdedict[hint] = rv
190          pdedict.update(failed_hints)
191          return pdedict
192  
193      else:
194          return _helper_simplify(eq, hints['hint'], hints['func'],
195                                  hints['order'], hints[hints['hint']], solvefun)

198  def _helper_simplify(eq, hint, func, order, match, solvefun):
199      """Helper function of pdsolve that calls the respective 
200      pde functions to solve for the partial differential 
201      equations. This minimizes the computation in 
202      calling _desolve multiple times. 
203      """
204      solvefunc = globals()["pde_" + hint.removesuffix("_Integral")]
205      return _handle_Integral(solvefunc(eq, func, order,
206          match, solvefun), func, order, hint)

209  def _handle_Integral(expr, func, order, hint):
210      r""" 
211      Converts a solution with integrals in it into an actual solution. 
212   
213      Simplifies the integral mainly using doit() 
214      """
215      if hint.endswith("_Integral"):
216          return expr
217  
218      elif hint == "1st_linear_constant_coeff":
219          return simplify(expr.doit())
220  
221      else:
222          return expr

225  def classify_pde(eq, func=None, dict=False, *, prep=True, **kwargs):
226      """ 
227      Returns a tuple of possible pdsolve() classifications for a PDE. 
228   
229      The tuple is ordered so that first item is the classification that 
230      pdsolve() uses to solve the PDE by default.  In general, 
231      classifications near the beginning of the list will produce 
232      better solutions faster than those near the end, though there are 
233      always exceptions.  To make pdsolve use a different classification, 
234      use pdsolve(PDE, func, hint=<classification>).  See also the pdsolve() 
235      docstring for different meta-hints you can use. 
236   
237      If ``dict`` is true, classify_pde() will return a dictionary of 
238      hint:match expression terms. This is intended for internal use by 
239      pdsolve().  Note that because dictionaries are ordered arbitrarily, 
240      this will most likely not be in the same order as the tuple. 
241   
242      You can get help on different hints by doing help(pde.pde_hintname), 
243      where hintname is the name of the hint without "_Integral". 
244   
245      See sympy.pde.allhints or the sympy.pde docstring for a list of all 
246      supported hints that can be returned from classify_pde. 
247   
248   
249      Examples 
250      ======== 
251   
252      >>> from sympy.solvers.pde import classify_pde 
253      >>> from sympy import Function, Eq 
254      >>> from sympy.abc import x, y 
255      >>> f = Function('f') 
256      >>> u = f(x, y) 
257      >>> ux = u.diff(x) 
258      >>> uy = u.diff(y) 
259      >>> eq = Eq(1 + (2*(ux/u)) + (3*(uy/u)), 0) 
260      >>> classify_pde(eq) 
261      ('1st_linear_constant_coeff_homogeneous',) 
262      """
263  
264      if func and len(func.args) != 2:
265          raise NotImplementedError("Right now only partial "
266              "differential equations of two variables are supported")
267  
268      if prep or func is None:
269          prep, func_ = _preprocess(eq, func)
270          if func is None:
271              func = func_
272  
273      if isinstance(eq, Equality):
274          if eq.rhs != 0:
275              return classify_pde(eq.lhs - eq.rhs, func)
276          eq = eq.lhs
277  
278      f = func.func
279      x = func.args[0]
280      y = func.args[1]
281      fx = f(x,y).diff(x)
282      fy = f(x,y).diff(y)
283  
284      # TODO : For now pde.py uses support offered by the ode_order function
285      # to find the order with respect to a multi-variable function. An
286      # improvement could be to classify the order of the PDE on the basis of
287      # individual variables.
288      order = ode_order(eq, f(x,y))
289  
290      # hint:matchdict or hint:(tuple of matchdicts)
291      # Also will contain "default":<default hint> and "order":order items.
292      matching_hints = {'order': order}
293  
294      if not order:
295          if dict:
296              matching_hints["default"] = None
297              return matching_hints
298          return ()
299  
300      eq = expand(eq)
301  
302      a = Wild('a', exclude = [f(x,y)])
303      b = Wild('b', exclude = [f(x,y), fx, fy, x, y])
304      c = Wild('c', exclude = [f(x,y), fx, fy, x, y])
305      d = Wild('d', exclude = [f(x,y), fx, fy, x, y])
306      e = Wild('e', exclude = [f(x,y), fx, fy])
307      n = Wild('n', exclude = [x, y])
308      # Try removing the smallest power of f(x,y)
309      # from the highest partial derivatives of f(x,y)
310      reduced_eq = eq
311      if eq.is_Add:
312          power = None
313          for i in set(combinations_with_replacement((x,y), order)):
314              coeff = eq.coeff(f(x,y).diff(*i))
315              if coeff == 1:
316                  continue
317              match = coeff.match(a*f(x,y)**n)
318              if match and match[a]:
319                  if power is None or match[n] < power:
320                      power = match[n]
321          if power:
322              den = f(x,y)**power
323              reduced_eq = Add(*[arg/den for arg in eq.args])
324  
325      if order == 1:
326          reduced_eq = collect(reduced_eq, f(x, y))
327          r = reduced_eq.match(b*fx + c*fy + d*f(x,y) + e)
328          if r:
329              if not r[e]:
330                  ## Linear first-order homogeneous partial-differential
331                  ## equation with constant coefficients
332                  r.update({'b': b, 'c': c, 'd': d})
333                  matching_hints["1st_linear_constant_coeff_homogeneous"] = r
334              elif r[b]**2 + r[c]**2 != 0:
335                  ## Linear first-order general partial-differential
336                  ## equation with constant coefficients
337                  r.update({'b': b, 'c': c, 'd': d, 'e': e})
338                  matching_hints["1st_linear_constant_coeff"] = r
339                  matching_hints["1st_linear_constant_coeff_Integral"] = r
340  
341          else:
342              b = Wild('b', exclude=[f(x, y), fx, fy])
343              c = Wild('c', exclude=[f(x, y), fx, fy])
344              d = Wild('d', exclude=[f(x, y), fx, fy])
345              r = reduced_eq.match(b*fx + c*fy + d*f(x,y) + e)
346              if r:
347                  r.update({'b': b, 'c': c, 'd': d, 'e': e})
348                  matching_hints["1st_linear_variable_coeff"] = r
349  
350      # Order keys based on allhints.
351      rettuple = tuple(i for i in allhints if i in matching_hints)
352  
353      if dict:
354          # Dictionaries are ordered arbitrarily, so make note of which
355          # hint would come first for pdsolve().  Use an ordered dict in Py 3.
356          matching_hints["default"] = None
357          matching_hints["ordered_hints"] = rettuple
358          for i in allhints:
359              if i in matching_hints:
360                  matching_hints["default"] = i
361                  break
362          return matching_hints
363      return rettuple

366  def checkpdesol(pde, sol, func=None, solve_for_func=True):
367      """ 
368      Checks if the given solution satisfies the partial differential 
369      equation. 
370   
371      pde is the partial differential equation which can be given in the 
372      form of an equation or an expression. sol is the solution for which 
373      the pde is to be checked. This can also be given in an equation or 
374      an expression form. If the function is not provided, the helper 
375      function _preprocess from deutils is used to identify the function. 
376   
377      If a sequence of solutions is passed, the same sort of container will be 
378      used to return the result for each solution. 
379   
380      The following methods are currently being implemented to check if the 
381      solution satisfies the PDE: 
382   
383          1. Directly substitute the solution in the PDE and check. If the 
384             solution has not been solved for f, then it will solve for f 
385             provided solve_for_func has not been set to False. 
386   
387      If the solution satisfies the PDE, then a tuple (True, 0) is returned. 
388      Otherwise a tuple (False, expr) where expr is the value obtained 
389      after substituting the solution in the PDE. However if a known solution 
390      returns False, it may be due to the inability of doit() to simplify it to zero. 
391   
392      Examples 
393      ======== 
394   
395      >>> from sympy import Function, symbols 
396      >>> from sympy.solvers.pde import checkpdesol, pdsolve 
397      >>> x, y = symbols('x y') 
398      >>> f = Function('f') 
399      >>> eq = 2*f(x,y) + 3*f(x,y).diff(x) + 4*f(x,y).diff(y) 
400      >>> sol = pdsolve(eq) 
401      >>> assert checkpdesol(eq, sol)[0] 
402      >>> eq = x*f(x,y) + f(x,y).diff(x) 
403      >>> checkpdesol(eq, sol) 
404      (False, (x*F(4*x - 3*y) - 6*F(4*x - 3*y)/25 + 4*Subs(Derivative(F(_xi_1), _xi_1), _xi_1, 4*x - 3*y))*exp(-6*x/25 - 8*y/25)) 
405      """
406  
407      # Converting the pde into an equation
408      if not isinstance(pde, Equality):
409          pde = Eq(pde, 0)
410  
411      # If no function is given, try finding the function present.
412      if func is None:
413          try:
414              _, func = _preprocess(pde.lhs)
415          except ValueError:
416              funcs = [s.atoms(AppliedUndef) for s in (
417                  sol if is_sequence(sol, set) else [sol])]
418              funcs = set().union(funcs)
419              if len(funcs) != 1:
420                  raise ValueError(
421                      'must pass func arg to checkpdesol for this case.')
422              func = funcs.pop()
423  
424      # If the given solution is in the form of a list or a set
425      # then return a list or set of tuples.
426      if is_sequence(sol, set):
427          return type(sol)([checkpdesol(
428              pde, i, func=func,
429              solve_for_func=solve_for_func) for i in sol])
430  
431      # Convert solution into an equation
432      if not isinstance(sol, Equality):
433          sol = Eq(func, sol)
434      elif sol.rhs == func:
435          sol = sol.reversed
436  
437      # Try solving for the function
438      solved = sol.lhs == func and not sol.rhs.has(func)
439      if solve_for_func and not solved:
440          solved = solve(sol, func)
441          if solved:
442              if len(solved) == 1:
443                  return checkpdesol(pde, Eq(func, solved[0]),
444                      func=func, solve_for_func=False)
445              else:
446                  return checkpdesol(pde, [Eq(func, t) for t in solved],
447                      func=func, solve_for_func=False)
448  
449      # try direct substitution of the solution into the PDE and simplify
450      if sol.lhs == func:
451          pde = pde.lhs - pde.rhs
452          s = simplify(pde.subs(func, sol.rhs).doit())
453          return s is S.Zero, s
454  
455      raise NotImplementedError(filldedent('''
456          Unable to test if %s is a solution to %s.''' % (sol, pde)))
457

460  def pde_1st_linear_constant_coeff_homogeneous(eq, func, order, match, solvefun):
461      r""" 
462      Solves a first order linear homogeneous 
463      partial differential equation with constant coefficients. 
464   
465      The general form of this partial differential equation is 
466   
467      .. math:: a \frac{\partial f(x,y)}{\partial x} 
468                + b \frac{\partial f(x,y)}{\partial y} + c f(x,y) = 0 
469   
470      where `a`, `b` and `c` are constants. 
471   
472      The general solution is of the form: 
473   
474      .. math:: 
475          f(x, y) = F(- a y + b x ) e^{- \frac{c (a x + b y)}{a^2 + b^2}} 
476   
477      and can be found in SymPy with ``pdsolve``:: 
478   
479          >>> from sympy.solvers import pdsolve 
480          >>> from sympy.abc import x, y, a, b, c 
481          >>> from sympy import Function, pprint 
482          >>> f = Function('f') 
483          >>> u = f(x,y) 
484          >>> ux = u.diff(x) 
485          >>> uy = u.diff(y) 
486          >>> genform = a*ux + b*uy + c*u 
487          >>> pprint(genform) 
488            d               d 
489          a*--(f(x, y)) + b*--(f(x, y)) + c*f(x, y) 
490            dx              dy 
491   
492          >>> pprint(pdsolve(genform)) 
493                                   -c*(a*x + b*y) 
494                                   --------------- 
495                                        2    2 
496                                       a  + b 
497          f(x, y) = F(-a*y + b*x)*e 
498   
499      Examples 
500      ======== 
501   
502      >>> from sympy import pdsolve 
503      >>> from sympy import Function, pprint 
504      >>> from sympy.abc import x,y 
505      >>> f = Function('f') 
506      >>> pdsolve(f(x,y) + f(x,y).diff(x) + f(x,y).diff(y)) 
507      Eq(f(x, y), F(x - y)*exp(-x/2 - y/2)) 
508      >>> pprint(pdsolve(f(x,y) + f(x,y).diff(x) + f(x,y).diff(y))) 
509                            x   y 
510                          - - - - 
511                            2   2 
512      f(x, y) = F(x - y)*e 
513   
514      References 
515      ========== 
516   
517      - Viktor Grigoryan, "Partial Differential Equations" 
518        Math 124A - Fall 2010, pp.7 
519   
520      """
521      # TODO : For now homogeneous first order linear PDE's having
522      # two variables are implemented. Once there is support for
523      # solving systems of ODE's, this can be extended to n variables.
524  
525      f = func.func
526      x = func.args[0]
527      y = func.args[1]
528      b = match[match['b']]
529      c = match[match['c']]
530      d = match[match['d']]
531      return Eq(f(x,y), exp(-S(d)/(b**2 + c**2)*(b*x + c*y))*solvefun(c*x - b*y))

534  def pde_1st_linear_constant_coeff(eq, func, order, match, solvefun):
535      r""" 
536      Solves a first order linear partial differential equation 
537      with constant coefficients. 
538   
539      The general form of this partial differential equation is 
540   
541      .. math:: a \frac{\partial f(x,y)}{\partial x} 
542                + b \frac{\partial f(x,y)}{\partial y} 
543                + c f(x,y) = G(x,y) 
544   
545      where `a`, `b` and `c` are constants and `G(x, y)` can be an arbitrary 
546      function in `x` and `y`. 
547   
548      The general solution of the PDE is: 
549   
550      .. math:: 
551          f(x, y) = \left. \left[F(\eta) + \frac{1}{a^2 + b^2} 
552          \int\limits^{a x + b y} G\left(\frac{a \xi + b \eta}{a^2 + b^2}, 
553          \frac{- a \eta + b \xi}{a^2 + b^2} \right) 
554          e^{\frac{c \xi}{a^2 + b^2}}\, d\xi\right] 
555          e^{- \frac{c \xi}{a^2 + b^2}} 
556          \right|_{\substack{\eta=- a y + b x\\ \xi=a x + b y }}\, , 
557   
558      where `F(\eta)` is an arbitrary single-valued function. The solution 
559      can be found in SymPy with ``pdsolve``:: 
560   
561          >>> from sympy.solvers import pdsolve 
562          >>> from sympy.abc import x, y, a, b, c 
563          >>> from sympy import Function, pprint 
564          >>> f = Function('f') 
565          >>> G = Function('G') 
566          >>> u = f(x, y) 
567          >>> ux = u.diff(x) 
568          >>> uy = u.diff(y) 
569          >>> genform = a*ux + b*uy + c*u - G(x,y) 
570          >>> pprint(genform) 
571            d               d 
572          a*--(f(x, y)) + b*--(f(x, y)) + c*f(x, y) - G(x, y) 
573            dx              dy 
574          >>> pprint(pdsolve(genform, hint='1st_linear_constant_coeff_Integral')) 
575                    //          a*x + b*y                                             \         \| 
576                    ||              /                                                 |         || 
577                    ||             |                                                  |         || 
578                    ||             |                                      c*xi        |         || 
579                    ||             |                                     -------      |         || 
580                    ||             |                                      2    2      |         || 
581                    ||             |      /a*xi + b*eta  -a*eta + b*xi\  a  + b       |         || 
582                    ||             |     G|------------, -------------|*e        d(xi)|         || 
583                    ||             |      |   2    2         2    2   |               |         || 
584                    ||             |      \  a  + b         a  + b    /               |  -c*xi  || 
585                    ||             |                                                  |  -------|| 
586                    ||            /                                                   |   2    2|| 
587                    ||                                                                |  a  + b || 
588          f(x, y) = ||F(eta) + -------------------------------------------------------|*e       || 
589                    ||                                  2    2                        |         || 
590                    \\                                 a  + b                         /         /|eta=-a*y + b*x, xi=a*x + b*y 
591   
592      Examples 
593      ======== 
594   
595      >>> from sympy.solvers.pde import pdsolve 
596      >>> from sympy import Function, pprint, exp 
597      >>> from sympy.abc import x,y 
598      >>> f = Function('f') 
599      >>> eq = -2*f(x,y).diff(x) + 4*f(x,y).diff(y) + 5*f(x,y) - exp(x + 3*y) 
600      >>> pdsolve(eq) 
601      Eq(f(x, y), (F(4*x + 2*y)*exp(x/2) + exp(x + 4*y)/15)*exp(-y)) 
602   
603      References 
604      ========== 
605   
606      - Viktor Grigoryan, "Partial Differential Equations" 
607        Math 124A - Fall 2010, pp.7 
608   
609      """
610  
611      # TODO : For now homogeneous first order linear PDE's having
612      # two variables are implemented. Once there is support for
613      # solving systems of ODE's, this can be extended to n variables.
614      xi, eta = symbols("xi eta")
615      f = func.func
616      x = func.args[0]
617      y = func.args[1]
618      b = match[match['b']]
619      c = match[match['c']]
620      d = match[match['d']]
621      e = -match[match['e']]
622      expterm = exp(-S(d)/(b**2 + c**2)*xi)
623      functerm = solvefun(eta)
624      solvedict = solve((b*x + c*y - xi, c*x - b*y - eta), x, y)
625      # Integral should remain as it is in terms of xi,
626      # doit() should be done in _handle_Integral.
627      genterm = (1/S(b**2 + c**2))*Integral(
628          (1/expterm*e).subs(solvedict), (xi, b*x + c*y))
629      return Eq(f(x,y), Subs(expterm*(functerm + genterm),
630          (eta, xi), (c*x - b*y, b*x + c*y)))

633  def pde_1st_linear_variable_coeff(eq, func, order, match, solvefun):
634      r""" 
635      Solves a first order linear partial differential equation 
636      with variable coefficients. The general form of this partial 
637      differential equation is 
638   
639      .. math:: a(x, y) \frac{\partial f(x, y)}{\partial x} 
640                  + b(x, y) \frac{\partial f(x, y)}{\partial y} 
641                  + c(x, y) f(x, y) = G(x, y) 
642   
643      where `a(x, y)`, `b(x, y)`, `c(x, y)` and `G(x, y)` are arbitrary 
644      functions in `x` and `y`. This PDE is converted into an ODE by 
645      making the following transformation: 
646   
647      1. `\xi` as `x` 
648   
649      2. `\eta` as the constant in the solution to the differential 
650         equation `\frac{dy}{dx} = -\frac{b}{a}` 
651   
652      Making the previous substitutions reduces it to the linear ODE 
653   
654      .. math:: a(\xi, \eta)\frac{du}{d\xi} + c(\xi, \eta)u - G(\xi, \eta) = 0 
655   
656      which can be solved using ``dsolve``. 
657   
658      >>> from sympy.abc import x, y 
659      >>> from sympy import Function, pprint 
660      >>> a, b, c, G, f= [Function(i) for i in ['a', 'b', 'c', 'G', 'f']] 
661      >>> u = f(x,y) 
662      >>> ux = u.diff(x) 
663      >>> uy = u.diff(y) 
664      >>> genform = a(x, y)*u + b(x, y)*ux + c(x, y)*uy - G(x,y) 
665      >>> pprint(genform) 
666                                           d                     d 
667      -G(x, y) + a(x, y)*f(x, y) + b(x, y)*--(f(x, y)) + c(x, y)*--(f(x, y)) 
668                                           dx                    dy 
669   
670   
671      Examples 
672      ======== 
673   
674      >>> from sympy.solvers.pde import pdsolve 
675      >>> from sympy import Function, pprint 
676      >>> from sympy.abc import x,y 
677      >>> f = Function('f') 
678      >>> eq =  x*(u.diff(x)) - y*(u.diff(y)) + y**2*u - y**2 
679      >>> pdsolve(eq) 
680      Eq(f(x, y), F(x*y)*exp(y**2/2) + 1) 
681   
682      References 
683      ========== 
684   
685      - Viktor Grigoryan, "Partial Differential Equations" 
686        Math 124A - Fall 2010, pp.7 
687   
688      """
689      from sympy.solvers.ode import dsolve
690  
691      eta = symbols("eta")
692      f = func.func
693      x = func.args[0]
694      y = func.args[1]
695      b = match[match['b']]
696      c = match[match['c']]
697      d = match[match['d']]
698      e = -match[match['e']]
699  
700  
701      if not d:
702           # To deal with cases like b*ux = e or c*uy = e
703          if not (b and c):
704              if c:
705                  try:
706                      tsol = integrate(e/c, y)
707                  except NotImplementedError:
708                      raise NotImplementedError("Unable to find a solution"
709                          " due to inability of integrate")
710                  else:
711                      return Eq(f(x,y), solvefun(x) + tsol)
712              if b:
713                  try:
714                      tsol = integrate(e/b, x)
715                  except NotImplementedError:
716                      raise NotImplementedError("Unable to find a solution"
717                          " due to inability of integrate")
718                  else:
719                      return Eq(f(x,y), solvefun(y) + tsol)
720  
721      if not c:
722          # To deal with cases when c is 0, a simpler method is used.
723          # The PDE reduces to b*(u.diff(x)) + d*u = e, which is a linear ODE in x
724          plode = f(x).diff(x)*b + d*f(x) - e
725          sol = dsolve(plode, f(x))
726          syms = sol.free_symbols - plode.free_symbols - {x, y}
727          rhs = _simplify_variable_coeff(sol.rhs, syms, solvefun, y)
728          return Eq(f(x, y), rhs)
729  
730      if not b:
731          # To deal with cases when b is 0, a simpler method is used.
732          # The PDE reduces to c*(u.diff(y)) + d*u = e, which is a linear ODE in y
733          plode = f(y).diff(y)*c + d*f(y) - e
734          sol = dsolve(plode, f(y))
735          syms = sol.free_symbols - plode.free_symbols - {x, y}
736          rhs = _simplify_variable_coeff(sol.rhs, syms, solvefun, x)
737          return Eq(f(x, y), rhs)
738  
739      dummy = Function('d')
740      h = (c/b).subs(y, dummy(x))
741      sol = dsolve(dummy(x).diff(x) - h, dummy(x))
742      if isinstance(sol, list):
743          sol = sol[0]
744      solsym = sol.free_symbols - h.free_symbols - {x, y}
745      if len(solsym) == 1:
746          solsym = solsym.pop()
747          etat = (solve(sol, solsym)[0]).subs(dummy(x), y)
748          ysub = solve(eta - etat, y)[0]
749          deq = (b*(f(x).diff(x)) + d*f(x) - e).subs(y, ysub)
750          final = (dsolve(deq, f(x), hint='1st_linear')).rhs
751          if isinstance(final, list):
752              final = final[0]
753          finsyms = final.free_symbols - deq.free_symbols - {x, y}
754          rhs = _simplify_variable_coeff(final, finsyms, solvefun, etat)
755          return Eq(f(x, y), rhs)
756  
757      else:
758          raise NotImplementedError("Cannot solve the partial differential equation due"
759              " to inability of constantsimp")
760

762  def _simplify_variable_coeff(sol, syms, func, funcarg):
763      r""" 
764      Helper function to replace constants by functions in 1st_linear_variable_coeff 
765      """
766      eta = Symbol("eta")
767      if len(syms) == 1:
768          sym = syms.pop()
769          final = sol.subs(sym, func(funcarg))
770  
771      else:
772          for sym in syms:
773              final = sol.subs(sym, func(funcarg))
774  
775      return simplify(final.subs(eta, funcarg))

778  def pde_separate(eq, fun, sep, strategy='mul'):
779      """Separate variables in partial differential equation either by additive 
780      or multiplicative separation approach. It tries to rewrite an equation so 
781      that one of the specified variables occurs on a different side of the 
782      equation than the others. 
783   
784      :param eq: Partial differential equation 
785   
786      :param fun: Original function F(x, y, z) 
787   
788      :param sep: List of separated functions [X(x), u(y, z)] 
789   
790      :param strategy: Separation strategy. You can choose between additive 
791          separation ('add') and multiplicative separation ('mul') which is 
792          default. 
793   
794      Examples 
795      ======== 
796   
797      >>> from sympy import E, Eq, Function, pde_separate, Derivative as D 
798      >>> from sympy.abc import x, t 
799      >>> u, X, T = map(Function, 'uXT') 
800   
801      >>> eq = Eq(D(u(x, t), x), E**(u(x, t))*D(u(x, t), t)) 
802      >>> pde_separate(eq, u(x, t), [X(x), T(t)], strategy='add') 
803      [exp(-X(x))*Derivative(X(x), x), exp(T(t))*Derivative(T(t), t)] 
804   
805      >>> eq = Eq(D(u(x, t), x, 2), D(u(x, t), t, 2)) 
806      >>> pde_separate(eq, u(x, t), [X(x), T(t)], strategy='mul') 
807      [Derivative(X(x), (x, 2))/X(x), Derivative(T(t), (t, 2))/T(t)] 
808   
809      See Also 
810      ======== 
811      pde_separate_add, pde_separate_mul 
812      """
813  
814      do_add = False
815      if strategy == 'add':
816          do_add = True
817      elif strategy == 'mul':
818          do_add = False
819      else:
820          raise ValueError('Unknown strategy: %s' % strategy)
821  
822      if isinstance(eq, Equality):
823          if eq.rhs != 0:
824              return pde_separate(Eq(eq.lhs - eq.rhs, 0), fun, sep, strategy)
825      else:
826          return pde_separate(Eq(eq, 0), fun, sep, strategy)
827  
828      if eq.rhs != 0:
829          raise ValueError("Value should be 0")
830  
831      # Handle arguments
832      orig_args = list(fun.args)
833      subs_args = [arg for s in sep for arg in s.args]
834  
835      if do_add:
836          functions = reduce(operator.add, sep)
837      else:
838          functions = reduce(operator.mul, sep)
839  
840      # Check whether variables match
841      if len(subs_args) != len(orig_args):
842          raise ValueError("Variable counts do not match")
843      # Check for duplicate arguments like  [X(x), u(x, y)]
844      if has_dups(subs_args):
845          raise ValueError("Duplicate substitution arguments detected")
846      # Check whether the variables match
847      if set(orig_args) != set(subs_args):
848          raise ValueError("Arguments do not match")
849  
850      # Substitute original function with separated...
851      result = eq.lhs.subs(fun, functions).doit()
852  
853      # Divide by terms when doing multiplicative separation
854      if not do_add:
855          eq = 0
856          for i in result.args:
857              eq += i/functions
858          result = eq
859  
860      svar = subs_args[0]
861      dvar = subs_args[1:]
862      return _separate(result, svar, dvar)

865  def pde_separate_add(eq, fun, sep):
866      """ 
867      Helper function for searching additive separable solutions. 
868   
869      Consider an equation of two independent variables x, y and a dependent 
870      variable w, we look for the product of two functions depending on different 
871      arguments: 
872   
873      `w(x, y, z) = X(x) + y(y, z)` 
874   
875      Examples 
876      ======== 
877   
878      >>> from sympy import E, Eq, Function, pde_separate_add, Derivative as D 
879      >>> from sympy.abc import x, t 
880      >>> u, X, T = map(Function, 'uXT') 
881   
882      >>> eq = Eq(D(u(x, t), x), E**(u(x, t))*D(u(x, t), t)) 
883      >>> pde_separate_add(eq, u(x, t), [X(x), T(t)]) 
884      [exp(-X(x))*Derivative(X(x), x), exp(T(t))*Derivative(T(t), t)] 
885   
886      """
887      return pde_separate(eq, fun, sep, strategy='add')

890  def pde_separate_mul(eq, fun, sep):
891      """ 
892      Helper function for searching multiplicative separable solutions. 
893   
894      Consider an equation of two independent variables x, y and a dependent 
895      variable w, we look for the product of two functions depending on different 
896      arguments: 
897   
898      `w(x, y, z) = X(x)*u(y, z)` 
899   
900      Examples 
901      ======== 
902   
903      >>> from sympy import Function, Eq, pde_separate_mul, Derivative as D 
904      >>> from sympy.abc import x, y 
905      >>> u, X, Y = map(Function, 'uXY')

910   
911      """
912      return pde_separate(eq, fun, sep, strategy='mul')

915  def _separate(eq, dep, others):
916      """Separate expression into two parts based on dependencies of variables."""
917  
918      # FIRST PASS
919      # Extract derivatives depending our separable variable...
920      terms = set()
921      for term in eq.args:
922          if term.is_Mul:
923              for i in term.args:
924                  if i.is_Derivative and not i.has(*others):
925                      terms.add(term)
926                      continue
927          elif term.is_Derivative and not term.has(*others):
928              terms.add(term)
929      # Find the factor that we need to divide by
930      div = set()
931      for term in terms:
932          ext, sep = term.expand().as_independent(dep)
933          # Failed?
934          if sep.has(*others):
935              return None
936          div.add(ext)
937      # FIXME: Find lcm() of all the divisors and divide with it, instead of
938      # current hack :(
939      # https://github.com/sympy/sympy/issues/4597
940      if len(div) > 0:
941          # double sum required or some tests will fail
942          eq = Add(*[simplify(Add(*[term/i for i in div])) for term in eq.args])
943      # SECOND PASS - separate the derivatives
944      div = set()
945      lhs = rhs = 0
946      for term in eq.args:
947          # Check, whether we have already term with independent variable...
948          if not term.has(*others):
949              lhs += term
950              continue
951          # ...otherwise, try to separate
952          temp, sep = term.expand().as_independent(dep)
953          # Failed?
954          if sep.has(*others):
955              return None
956          # Extract the divisors
957          div.add(sep)
958          rhs -= term.expand()
959      # Do the division
960      fulldiv = reduce(operator.add, div)
961      lhs = simplify(lhs/fulldiv).expand()
962      rhs = simplify(rhs/fulldiv).expand()
963      # ...and check whether we were successful :)
964      if lhs.has(*others) or rhs.has(dep):
965          return None
966      return [lhs, rhs]
