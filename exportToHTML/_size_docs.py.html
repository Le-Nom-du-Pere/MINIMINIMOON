14   numel() -> int 
15    
16   Returns the number of elements a :class:`torch.Tensor` with the given size would contain. 
17    
18   More formally, for a tensor ``x = tensor.ones(10, 10)`` with size ``s = torch.Size([10, 10])``, 
19   ``x.numel() == x.size().numel() == s.numel() == 100`` holds true. 
20    
21   Example:: 
22    
23       >>> x=torch.ones(10, 10) 
24       >>> s=x.size() 
25       >>> s 
26       torch.Size([10, 10]) 
27       >>> s.numel() 
28       100 
29       >>> x.numel() == s.numel() 
30       True 
31    
32    
33   .. warning:: 
34    
35       This function does not return the number of dimensions described by :class:`torch.Size`, but instead the number 
36       of elements a :class:`torch.Tensor` with that size would contain. 
37
